Git基本命令集

2016-11-23
工具
git

本文介绍运用git将代码同步到github上的基本方法。

  • 检验电脑上是否存在 ssh keys.

  • 创建新的ssh keys

  • 输入两次密码 tyrival1019（与Github一致）
  • 进入C:\User\用户名.ssh，打开 id_rsa.pub，将其作为一个SSH key新建到Github中的个人信息中

  • 配置用户名、邮箱（与github保持一致）

  • 在Github上新建项目，并复制其ssh地址

  • 进入项目文件夹，初始化仓库.

  • 添加所有文件到仓库.

  • 提交.

  • 远程同步到服务器

  • 第5步如果报“fatal: remote origin already exists.”

  • 推送到服务器
